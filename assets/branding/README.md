# SATORI Studio branding assets

This directory contains the runtime branding assets derived from the Phase B canonical SVG specification.

Included files:
- satori-logo.svg
- satori-icon.svg
- satori-menu-icon.svg
- satori-studio-menu.svg
- satori-studio-lite-menu.svg
- satori-plugin-icon.png

Menu icon notes:
- Studio Lite branding is SVG-only for runtime usage.
- `satori-icon.svg` is the canonical Studio Lite SVG, applied directly.
- `satori-menu-icon.svg` is the path-stable WordPress admin menu icon (optical variant of the Studio Lite mark).
- `satori-studio-menu.svg` and `satori-studio-lite-menu.svg` are optical-size menu variants and should not replace canonical SVG sources.
- PNG assets are intentionally not used for Studio Lite runtime branding.
