# SATORI Studio

SATORI Studio – Lite is a GPL-licensed fork of Beaver Builder Lite.

## Project Foundation

- The core builder engine originates from Beaver Builder Lite.
- All original Beaver Builder copyright remains with Beaver Builder authors.
- SATORI Studio – Lite is focused on:
  - Operational clarity
  - Workflow efficiency
  - Governance
  - Production stability
- SATORI Studio – Pro (SSP) is the commercial extension layer built on top of the Lite foundation.

## Positioning

SATORI Studio – Lite provides a stable, curated builder foundation.
SATORI Studio – Pro expands functionality through modular commercial extensions.

## Licensing

- Lite remains GPL-compliant and attribution-forward.
- The Lite fork is not monetised as a repackaged Beaver Builder Lite product.
- Commercial value is delivered through SSP extension modules and platform services.
